## Study Guide by Difficulty

### Easy Problems (Start Here)
Focus on these to build confidence:
- Vector operations ([Problem 5](./dirty_problems/problem_5.md))
- Basic gradient calculations (Group 3.1)
- Basic line integrals with simple parameterization (Group 4.1)
- Area calculations (Group 4.6)

### Medium Problems (Core Exam Material)
Most exam problems will be at this level:
- Directional derivatives (Group 3.2)
- Finding scalar functions from gradients (Group 3.3)
- Divergence and curl calculations (Groups 3.4, 3.5)
- Line integrals along multiple paths (Group 4.2)
- Work done problems (Group 4.5)
- Surface problems (Part V)

### Hard Problems (Advanced Practice)
Challenge yourself with:
- Vector identity proofs (Group 1.2)
- Complex line integrals (Group 4.4)
- Laplacian calculations (Group 3.6)
