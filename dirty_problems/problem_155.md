# Problem 155: Continuation of Line Integral Evaluation

## Problem Statement

This page contains continuation of calculations for evaluating line integrals along different paths, showing the final steps for parts (c) and (d).

## Solution

### Part (c): Final Calculation

From the previous page, the integral along straight line segments:

**First segment:** $(1,1)$ to $(1,2)$:
$$\int_1^2 3 dy = 3$$

**Second segment:** $(1,2)$ to $(4,2)$:
$$\int_1^4 (x + 2) dx = 13.5$$

**Total:**
$$3 + 13.5 = 16.5 = \frac{33}{2}$$

### Part (d): Final Calculation

From the parametric curve with $x = 2t^2 + t + 1$, $y = t^2 + 1$:

The integral simplifies to:

$$\int_0^1 [8t^3 + 4t^2 + 15t + 2] dt$$

$$= \left[2t^4 + \frac{4}{3}t^3 + \frac{15}{2}t^2 + 2t\right]_0^1$$

$$= 2 + \frac{4}{3} + \frac{15}{2} + 2 = \frac{32}{3}$$

## Answer

**Part (c):**

$$\boxed{\frac{33}{2}}$$

**Part (d):**

$$\boxed{\frac{32}{3}}$$
